# Rocket Launch Portfolio

Interactive Three.js portfolio for Farouk El Hammoumi.

## Run Locally

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4173`.

The Three.js runtime is stored locally in `vendor/three.module.js`.
The control-room monitor model and the `need_some_space.glb` hologram are local assets, so use a local server or GitHub Pages instead of opening `index.html` with `file://`.
